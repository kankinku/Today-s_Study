# [Database] 화자 분석 패턴 로그 (Analysis Pattern Log)

이 문서는 개별 분석 보고서에서 추출한 화자의 '분석 패턴(Thinking Pattern)'을 데이터베이스화하여 기록하는 로그 파일입니다. 각 문서는 **주제(Topic)**, **관점(Lens)**, **논리 연결(Connectivity)**, **추론 방식(Inference Logic)**으로 분해되어 저장됩니다.
